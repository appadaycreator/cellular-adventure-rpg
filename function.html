プロジェクト概要

プロジェクト名

細胞の冒険 - Cellular Adventure

プロジェクト概要

ライフゲームをベースにした教育的シミュレーションRPG。 生物学の基礎概念を楽しく学べるWebアプリケーション。

開発目標

  • • 生物学の基礎概念を視覚的に学習
  • • インタラクティブなシミュレーション体験
  • • レスポンシブデザインによる全デバイス対応
  • • PWA機能によるオフライン動作

技術スタック

HTML5 CSS3 JavaScript ES6+ PWA Responsive

ライブラリ

  • • Tailwind CSS (v2.2.19)
  • • Font Awesome (v6.4.0)
  • • Google Fonts

コア機能

ライフゲームシミュレーション

Conway's Game of Lifeの完全実装。細胞の誕生・生存・死亡のルールに基づくシミュレーション。

実装済み v1.0.0

インタラクティブグリッド

クリックで細胞の配置・削除が可能。リアルタイムでグリッドの状態を編集。

実装済み v1.0.0

速度調整

シミュレーション速度を1-10段階で調整。観察に最適な速度設定。

実装済み v1.0.0

グリッドサイズ変更

20x15〜50x40のサイズ選択。用途に応じたグリッドサイズの調整。

実装済み v1.0.0

プリセットパターン

グライダー、振動子、静物、ガンなどの有名なパターンをワンクリックで配置。

実装済み v1.0.0

ランダム生成

ランダムな初期パターンを生成。予期しない発見を促進。

実装済み v1.0.0
